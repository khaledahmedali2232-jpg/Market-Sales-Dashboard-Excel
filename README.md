# Market Sales Dashboard | Excel

## Overview

This project presents an interactive sales dashboard built using Microsoft Excel to analyze sales performance and customer behavior.

## Dashboard Features

* Total Sales Analysis
* Total Orders
* Average Quantity
* Sales by Category
* Orders by Size
* Monthly Sales Trends
* Gender Analysis
* Age Group Analysis
* Regional Sales Analysis
* Interactive Filters and Slicers

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Data Cleaning
* Dashboard Design

## Key Metrics

* Total Sales: $21.17M
* Total Orders: 28,471
* Average Quantity: 8


*   ## Key Insights

* Women accounted for 69% of total orders.
* Adults represented 50% of customers.
* Sets and Kurtas generated the highest sales.
* Medium (M) size had the largest number of orders.
* Sales peaked during March.
